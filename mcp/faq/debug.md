Das findest du in VS Code hier:

1. Menü **View** → **Output** (oder Shortcut `Strg+Shift+U`).
2. Unten öffnet sich das **Output Panel**.
3. Rechts oben im Dropdown kannst du den Kanal auswählen → dort gibt’s dann **Cursor MCP**.

👉 Wenn du es nicht siehst: evtl. ist das Extension-spezifisch (z. B. Cursor oder MCP Plugin), dann taucht der Eintrag erst auf, sobald das Plugin etwas ins Output schreibt.
